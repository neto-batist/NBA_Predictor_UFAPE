## NBA Predictor UFAPE: Predição de Dados e Resultados da NBA  

O NBA Predictor UFAPE é um projeto acadêmico desenvolvido na Universidade Federal do Agreste de Pernambuco (UFAPE), como parte da disciplina de Ciência de Dados, sob a orientação do professor Ryan Azevedo. O projeto é conduzido pelos alunos Davy Eugênio e João Batista Neto, com o objetivo de aplicar técnicas avançadas de análise e predição de dados para prever resultados de partidas da NBA (National Basketball Association).  

A proposta do projeto envolve a coleta, processamento e análise de dados estatísticos das temporadas passadas da NBA, utilizando ferramentas como Python, Pandas, Matplotlib e Machine Learning. O sistema desenvolvido busca identificar padrões de desempenho das equipes e jogadores, gerando insights valiosos sobre fatores determinantes para vitórias e derrotas.  

Dentre as funcionalidades do NBA Predictor UFAPE, destacam-se:  

- Análise estatística das partidas: Utilização de tabelas e gráficos para explorar métricas como média de pontos, aproveitamento em casa e fora de casa, e sequência de vitórias e derrotas.  
- Visualização de dados: Gráficos de barras, histograma, radar e dispersão para melhor compreensão das tendências de desempenho dos times.  
- Predição de resultados: Aplicação de modelos preditivos para estimar o vencedor de partidas futuras com base em dados históricos.  

Este projeto não apenas reforça a aplicação prática de conceitos de Ciência de Dados, mas também demonstra o potencial da análise estatística no esporte, contribuindo para estudos na área de analytics esportivo. O NBA Predictor UFAPE representa um avanço na interseção entre tecnologia e esportes, fornecendo uma plataforma acadêmica robusta para análise de desempenho da NBA.
